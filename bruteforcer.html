Computer Password Brute Forcer

DISCLAIMER: This project is for educational use only and is designed to demonstrate how brute-force techniques work in a controlled, offline environment.

This program simulates a 4-digit PIN brute-force attack on a local test account. It cannot access or affect any remote or real systems.

How to use:
1. Run "brute force.bat" on your own computer only.
2. Enter your IPv4 (private IP) address and computer username.
3. Choose a local wordlist - numspam.txt or passlist.txt - to simulate attempts.
4. The program will attempt to find the correct PIN on the local machine that this program is being run on.
5. This program will NOT remotely run on a computer, ONLY on the local machine it is run on and other machines on the same network.

Please do not use or adapt this tool for real systems. Doing so may be illegal under the Computer Misuse Act 1990.
